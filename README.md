# TileCreator
## _A tilemap editor because I'm bored_

TileCreator is a C++ application which will allow you to edit and create level for your games.
Qt-powered.

## Features

- None

## Installation

TileCreator provides a script which will setup the entire build for you.

```sh
chmod +x setup.sh
./setup.sh
```
